# Trabalho LP
  Dupla:
  
  Bruno Godoy
  Felipe Oshiro
  
 # Descrição
 
  O Jogo consiste em você controlar dois robozinhos para fugirem do labirinto e voltarem para casa.
  
  # Controles
  
  WASD - Movimento personagem 1
  
  Space - Tiro personagem 1
  
  Arrows Keys - Movimento personagem 2
  
  L - Tiro personagem 2
